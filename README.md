# DEVIL SHARABI - Message Automation Flask App

This is a Flask-based web application designed to send messages to a Facebook thread using Facebook API tokens. The app allows users to send messages at specified time intervals using single or multiple tokens. It also supports stopping the ongoing tasks for a specific thread. The app has a stylish user interface and provides links to WhatsApp and Facebook for easy access.

## Features:
- **Send messages**: Send messages using a single token or multiple tokens.
- **Stop messages**: Stop the ongoing messages being sent to a specific thread.
- **Stylish UI**: Custom CSS and Bootstrap are used to provide a modern and attractive UI.
- **WhatsApp & Facebook links**: Direct links for WhatsApp and Facebook integration.
- **Footer**: Displaying a footer with the branding of Mr. Devil 2025.

## Requirements:
- Python 3.7 or higher
- Flask
- Requests library
